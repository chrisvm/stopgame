Stop Game
=========

![GitHub Logo](public/images/StopLogo_300x300.png)

## Install 

To install, just run 

	npm install

in the root directory

## Usage 

To serve the webapp, just run `/bin/www` with node

	node bin/www 

This will start the webapp in localhost:3000 if no PORT variable is set.
